COMPONENT: Command Palette (Search)
================================================================================
File: components/CommandPalette.js

Type: Modal overlay / Command palette (Spotlight-style)
Trigger: Ctrl+K keyboard shortcut OR clicking search icon in navbar
Close: ESC key or clicking backdrop

Layout:
- Full-screen semi-transparent overlay
- Centered panel with glass morphism styling

Header:
- Search icon (magnifying glass)
- Text input: "Search products, pages, categories (e.g., 'abc', 'extinguisher')..."
- ESC keyboard shortcut badge

Results Area (when query is empty):
- "Quick Navigation" section header
- Quick links with icons:
  - 🏠 Home → /
  - 📦 Products → /products
  - 🔧 AMC Services → /amc
  - 🏢 About Us → /about
  - 📞 Contact → /contact

Results Area (when query has matches):
- "Smart Results" section header (gold)
- Up to 8 fuzzy-matched results, each showing:
  - Icon (emoji based on type)
  - Title (bold)
  - Subtitle (description or route)
  - Type badge: "page", "category", or "product" (gold badge)

Results Area (no matches):
- 🤖 emoji
- "Our smart search couldn't find anything for '{query}'"
- "Try broader terms or check spelling."

Footer:
- Keyboard shortcut hints: ↑↓ navigate | ↵ select | esc close

Search Sources:
- Pages (Home, Products, AMC Services, About Us, Contact)
- Product categories (6 categories)
- All products from Firestore (name, code, category, description)
- Uses Fuse.js for fuzzy matching (threshold: 0.4)

Keyboard Navigation:
- Arrow Up/Down to navigate results
- Enter to select/navigate
- ESC to close
